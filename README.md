# LinkedInvite
Script JS send invites automatically


1. Open Browser '[LinkedIn search people](https://www.linkedin.com/search/results/people/)'
2. Open the JavaScript console
3. Copy the content of invite.v2.js and paste it in the JavaScript console
4. Click on enter
5. 
```javascript
Linkedin.start() // start send invites automatically
Linkedin.count() // number of the invites
Linkedin.stop() // stop send invites automatically
```
